# ⚔️ OSRS Bank Layout Simulator 🏦

A web-based tool designed for Old School RuneScape players to plan, visualize, and save their bank layouts. <br>
Perfect for organizing gear setups, slayer tabs, or just achieving that peak bank aesthetic — without moving items manually in-game.

🔗 **Live Site:** [OSRS Bank Layout Simulator](https://pirrkatt.github.io/osrs-bank-sim/)

---

## ✨ Features

- **Drag & Drop Interface**  
  Smoothly move items between the panel and the bank grid.

- **Search Panel**  
  Quickly find any items using the built-in search bar.

- **Multiple Layout Tabs**  
  Create up to 10 unique tabs for different versions or different setups. *(PvM, Farming, Skilling, etc.)*

- **Swap & Insert Modes**  
  Toggle between swapping item positions or inserting them into the grid — just like in OSRS.

- **Auto-Save**  
  Your tabs are automatically saved in your browser using local storage.

- **Sharing (Import / Export)**  
  Use the Import and Export buttons to share your tabs with others.
