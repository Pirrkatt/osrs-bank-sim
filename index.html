OSRS Bank Simulator

How to use

  • Add & Remove Items: Drag & Drop items from the panel to the bank.
    Double-click an item to add it to the first available slot.
    Right-click items in the bank to remove them.
  • Swap/Insert: Use the Swap and Insert buttons in the bank to change how items behave when placed.
  • Tabs: Press the + button to create a new empty tab.
    Right-click on a tab to Save to it or Delete it.
  • Auto-Save: When activated, every move is saved to the active tab.
  • Feedback: Found a bug or have a feature idea?
    Open an issue on GitHub below!
